<objective>
Walking Skeleton for the GSD Governance Overlay. Stand up the project's first source tree, build tooling, and the `governance` CLI, then prove the thinnest end-to-end slice: author ONE real rule under `aidlc-rules/`, run `governance build-index`, and emit a valid, body-free `rule-index.json` — verified by an automated smoke test that runs the built CLI.
