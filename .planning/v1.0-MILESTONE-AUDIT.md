## Tech Debt (non-blocking, deferred)

### Phase 5 — Audit-Artifact Writer (8 items, from `05-REVIEW.md`)

The deep code review found 1 critical (CR-01: missing `matchedAxis` enum validation) — **resolved** in commit `d301c53` during this session's execute-phase run. The following advisory items remain open:

- **WR-01**: `assertTimestamp` uses `Date.parse` which accepts non-ISO formats — the "ISO timestamp" claim is not enforced
- **WR-02**: `selector_reason` typed as `SkipReason` but stored from `rule.reason` checked only as string; validation split across two functions with different error shapes
- **WR-03**: consent integration test never renders `verify:post` — consent-revocation→audit-hook cross-cutting has zero coverage despite `onError:halt` (the most dangerous silent-failure mode)
- **WR-04**: `isDirectRun` matches any file named `audit-artifact.js`, not just the dist entry (fragile)
- **WR-05**: `atomicWriteText` fixed `.tmp` suffix — concurrent writers to the same `GOVERNANCE.md` would race (audit-trail integrity risk under concurrency; v1 is single-writer so not currently reachable)
- **IN-01**: `buildAuditRecord` export wider than needed
- **IN-02**: `writeGovernanceAudit` returns input `outputPath`, not the resolved absolute path actually written
- **IN-03**: `resolveGsdTools` `as string` cast hides a theoretical `undefined` fallback

### Cross-phase config (1 item, from integration checker)

- `.planning/config.json` is shared across capabilities and carries unrelated keys (`tavily_search`, `ref_search`, `perplexity`, `jina`, `quick_branch_template`) that the GSD runtime warns about on every invocation. `governance.token_budget` is not set (default 2000 applies; defensive reader handles absence). Config namespacing would clean this up in a future phase.
